# 潮流好物 - 移动端电商 H5

纯原生 HTML/CSS/JS 移动端电商前端项目，无构建工具。

## 功能页面

| 页面 | 说明 |
|------|------|
| 首页 | 轮播图、分类入口、秒杀专区、热门推荐 |
| 分类 | 二级分类导航、商品列表 |
| 购物车 | 商品勾选、数量调整、批量删除 |
| 商品详情 | 图片轮播、规格选择、加入购物车 |
| 确认订单 | 地址选择、优惠券、支付方式 |
| 我的订单 | 订单列表、状态筛选 |
| 个人中心 | 收藏、足迹、设置、VIP |
| 收货地址 | 地址管理、增删改 |
| 优惠券 | 优惠券领取和使用 |

## 本地预览

```bash
npx serve pages -l 3000
```

访问 http://localhost:3000

## 部署到 Vercel

推送后自动部署，无需构建步骤。
